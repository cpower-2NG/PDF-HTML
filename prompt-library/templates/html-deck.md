# HTML 生成版

```text
请按照“先大纲、后 HTML”的方式，生成一套可导出 PDF 且尽量可转换为 PPT 的 HTML 演示文稿。

主题：
[主题]

受众：
[受众]

目标：
[希望观众理解什么、行动什么、记住什么]

必须包含的内容：
- [要点 1]
- [要点 2]
- [要点 3]

页面要求：
- 内容充实，每页一个主观点
- 有清晰的叙事顺序
- 适合 16:9 演示
- 适合浏览器打印成 PDF
- 尽量保持 PPT 转换友好
- 避免过度依赖绝对定位、滚动、悬浮和复杂动画
- 使用语义化 HTML 和 CSS 变量

输出顺序：
1. 概念摘要
2. 页面大纲
3. 完整 HTML
4. 导出说明
5. 如果需要，补充 PPT 转换建议

如果信息不足，请先列出需要我补充的最少问题。
```
